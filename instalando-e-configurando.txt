Para instalar o Git, basta acessar a URL https://git-scm.com/downloads e fazer o download conforme a versão do seu sitema Operacional.
Uma vez baixado e instalado, faça uma pré configuração com seus dados de autor: Crie uma pasta desejada para utilizar o git e, com o botão direito do mouse, selecione a opção GIT BASH HERE. Isso irá abrir o prompt de comando do Git.
Você pode utilizar o terminal de comando de sua IDE como neste caso, uso o VSCode.
- Para configurar seu nome digite o comando a seguir substituindo as aspas:
git config --global user.name "Seu Nome" [clique enter]
- Para configurar seu e-mail digite o comando:
git config --global user.email "seuemail@mail.com"

Pronto, seu GIT está pré configurado para usar.

Para a configuração do GitHub, acesse o site https://github.com e crie uma conta grátis inserindo todos seus dados necessários.
Ao acessar seu perfil, na area superior direita da tela você consegue inserir um novo repositório "New Repository" (que será sua biblioteca online de um determinado projeto)
- Dê um nome ao repositório
- Insira uma descrição breve sobre o que se trata o projeto
- Clique em Create Repository
Ao final, uma tela irá lhe fornecer instruções para sincronizar seu Git com o repositório criado no GitHub (Veja mais comandos no arquivo "principais-comandos").
